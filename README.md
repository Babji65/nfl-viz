# NFL Data Visualizations

An interactive data visualization project exploring NFL statistics and trends.

## Project Overview

This site presents visual analyses of NFL data including player performance, team statistics, and game trends — built with modern web technologies and designed to make complex data easy to explore.

## Tech Stack

- **HTML / CSS / JavaScript** — core site structure and styling
- **D3.js / Plotly.js** — interactive data visualizations
- **GitHub Pages** — free static hosting

## Getting Started

### Prerequisites
- A modern web browser
- (Optional) [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VS Code extension for local development

### Run Locally

```bash
git clone https://github.com/yourusername/nfl-viz.git
cd nfl-viz
# Open index.html in your browser, or use Live Server
```
