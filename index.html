The Project
About BallIQ

What We Built

BallIQ is a data science project focused on making NFL analytics accessible and visual. We collect, clean, and transform raw game data into interactive charts and dashboards that reveal patterns hidden in the numbers.

The Data

Our dataset spans multiple NFL seasons, covering play-by-play events, player performance metrics, team efficiency ratings, and advanced stats like EPA (Expected Points Added) and DVOA.

Tech Stack

Built with vanilla HTML, CSS, and JavaScript. Visualizations powered by Chart.js and D3.js. Data processed in Python using pandas and nfl_data_py. Hosted on GitHub Pages.
